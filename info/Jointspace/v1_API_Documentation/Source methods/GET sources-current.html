REST API Method: GET sources/current

Returns the id of the current source.

URL

http://ip-address:1925/1/sources/current

Format

json

HTTP Method

GET

Requires Authentication

false

Parameters

    none

Response

This resource returns a JSON object with key/value pairs. See the JSON example for how these key/value pairs are returned.

Keys

  • id (string): The id of the current source.

JSON example

{
    "id": "hdmi1"
}
