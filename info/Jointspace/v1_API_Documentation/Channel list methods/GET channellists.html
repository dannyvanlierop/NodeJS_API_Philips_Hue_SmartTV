REST API Method: GET channellists

Returns the id's of all channel lists.

URL

http://ip-address:1925/1/channellists

Format

json

HTTP Method

GET

Requires Authentication

false

Parameters

    none

Response

This resource returns a JSON object with key/value pairs. See the JSON example for how these key/value pairs are returned.

Keys

  • id (string): The id's of the channel lists are used as keys in the top level object. The value corresponding to the key is an object with details about the channel list.
  • name (string): The name of the channel list.
  • source (string): Either "tv" for TV channels or "satellite" for satellite channels.
